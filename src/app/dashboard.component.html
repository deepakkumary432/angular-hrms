Dear ladies and gentlemen,

In internal audits we discovered weak spot which we are consequently closing now.
One finding is related to the emergency procedure of answering three security questions for login.

From April 1st 2019 on, only the following login methods will be available for AVL desktops/notebooks running Windows:

  1. USB-Token/Smartcard -> Primary authentication.
  2. Offentry mobile application -> Alternative authentication.

This applies to all user accounts with strong authentication regardless of your current location (within the AVL network, at home, on site with customer etc).

Logging in by answering the Evidian Security Questions will not be an option as of April 1st 2019

Further information can be found in the FAQs on this topic (https://desktop.avl.com/sta). Should you have any Question, please contact the IT Helpdesk(+43-316-787-512).
